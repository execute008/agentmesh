# Phase 1, Plan 1: Foundry Project Scaffold

## Goal
`agentmesh-contracts/` is a correctly-configured Foundry project with OpenZeppelin v5 installed — a clean build environment Phase 1.2 can write Solidity contract code into immediately.

## Context
- Foundry default structure: `src/`, `test/`, `script/`, `lib/`
- OpenZeppelin v5.x via `forge install OpenZeppelin/openzeppelin-contracts` (HEAD, git submodule)
- Solidity `^0.8.20`, optimizer enabled at 200 runs
- No frontend, no npm — pure forge/git-submodule workflow
- Remove boilerplate `Counter.sol` / `Counter.t.sol` to keep project clean for Phase 1.2
- `forge init` must run inside the existing git repo (no `--no-git` flag)
