# Phase 2, Plan 1: AgentRegistry.sol Implementation

## Goal
`AgentRegistry.sol` is a fully implemented, compilable Foundry contract with agent registration, capability search, ETH escrow lifecycle, reputation tracking, NatSpec documentation, and all domain events emitted on state changes.

## Context
- The Foundry project lives at `agentmesh-contracts/` with `solc 0.8.20`, optimizer enabled, and a single remapping `@openzeppelin/contracts/=lib/openzeppelin-contracts/contracts/`.
- `src/` is empty (only `.gitkeep`). `test/` has a placeholder `Scaffold.t.sol`.
- The domain model defines two bounded contexts inside the same contract: **Registry** (agent identity, capabilities, endpoints) and **Escrow & Settlement** (task creation, completion, payment release, reputation).
- No existing interfaces or contracts to worry about — greenfield implementation.
- OpenZeppelin is available via `lib/openzeppelin-contracts` but this contract needs no OZ imports — pure custom logic with `require` strings.
- Design decisions: `agentId: string`, `capabilities: string[]`, `taskId: uint256` caller-supplied, reputation starts at 50 max 100 +5 per releasePayment, `releasePayment(taskId, requester)` requires `requester == msg.sender`, `completeTask` requires `msg.sender == task.executor`, all errors as `require(condition, "message")` string literals.

    ```solidity
    function searchByCapability(string calldata cap) external view returns (address[] memory) {
        uint256 count = 0;
        for (uint256 i = 0; i &lt; _agentList.length; i++) {
            Agent storage agent = _agents[_agentList[i]];
            for (uint256 j = 0; j &lt; agent.capabilities.length; j++) {
                if (keccak256(bytes(agent.capabilities[j])) == keccak256(bytes(cap))) {
                    count++;
                    break;
                }
            }
        }
        address[] memory result = new address[](count);
        uint256 idx = 0;
        for (uint256 i = 0; i &lt; _agentList.length; i++) {
            Agent storage agent = _agents[_agentList[i]];
            for (uint256 j = 0; j &lt; agent.capabilities.length; j++) {
                if (keccak256(bytes(agent.capabilities[j])) == keccak256(bytes(cap))) {
                    result[idx++] = _agentList[i];
                    break;
                }
            }
        }
        return result;
    }
    ```

    ```solidity
        Task storage task = _tasks[taskId];
        require(msg.sender == task.executor, "Only executor");

        task.completed = true;
        emit TaskCompleted(taskId);
    ```

    **releasePayment:**
    ```solidity
    function releasePayment(uint256 taskId, address requester) external {
        require(requester == msg.sender, "Only requester");
        Task storage task = _tasks[taskId];
        require(task.requester == requester, "Not task requester");
        require(task.completed, "Task not completed");
        require(!task.released, "Already released");
        task.released = true;
        uint256 amount = task.escrowAmount;

        Agent storage executor = _agents[task.executor];
        uint8 newRep = executor.reputation + 5;
        if (newRep > 100) newRep = 100;
        executor.reputation = newRep;

        (bool success, ) = payable(task.executor).call{value: amount}("");
        require(success, "Transfer failed");

        emit PaymentReleased(taskId, amount);
        emit ReputationUpdated(task.executor, newRep);
    }
    ```
